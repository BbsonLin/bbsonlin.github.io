早期

  1. 1992 - NCSA(國家超級電腦應用中心) Mosaic 最早流行的圖形介面網頁瀏覽器
  2. 1994 - Netscape(網景) 雇用了許多原來的 NCSA Mosaic 開發者用來開發 Mosaic Netscape(後改名為 Netscape Navigator)
  3. 1995 - Microsoft 取得Mosaic的授權後，推出了 Internet Explorer 1 (IE 1)

第一次瀏覽器大戰 (1995 - 2001)

Netscape(Netscape Navigator) vs Microsoft(Internet Explorer)

Netscape 方

  1. 1995 - Marc Lowell Andreessen 認為 HTML 需要一種膠水語言，後 Brendan Eich 花十天開發出 JavaScript 原型(起初稱 Mocha 和 LiveScript)
  2. 1996 - Netscape 提交 ECMA(歐洲電腦製造商協會) 提交語言標準，此標準後稱 ECMAScript (ES)
  3. 1997 - ECMA 以 JavaScript 語言為基礎制定了 ECMAScript 標準規範 ECMA-262
  4. 1998 - Netscape Navigator 開放原始碼，後由 Mozilla 組織維護

Microsoft 方

  1. 1996 - 微软模仿 JavaScript 开发了一种相近的语言，取名为 JScript，並用於 IE 3.0，成為第一款支援程式語言及CSS的商用瀏覽器
  2. 1997 - IE4.0 更符合 W3C 製定的網頁標準方面
  3. 1998 - Microsoft Windows 壟斷了作業系統市場，而 IE 與 Windows 的捆綁，讓使用者不再下載 Netscape，而改用 IE

第二次瀏覽器大戰 (2004 - 2017)

  1. 2003 年後 IE 市佔率逐漸受到其他瀏覽器蠶食。
  2. Opera(1996)、Safari(2003)、Firefox(2004)、Chrome(2008) 相繼出現。
  3. 2010 年後 HTML5、CSS3 等新標準的制定和智慧型手機、平板電腦的普及，網路瀏覽器的競爭也在新的領域展開。
